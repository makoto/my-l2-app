# L2 Resolver demo

## Pre-requisite

- Setup a gateway server by folllowing https://github.com/corpus-io/ENS-Bedrock-Resolver
- Register a name for the test on Goerli

## Setting up the server

```
gh repo clone makoto/my-l2-app
cd my-l2-app
yarn
yarn start
```
